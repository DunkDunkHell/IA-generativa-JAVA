#  Projeto: IA Generativa aplicada ao aprendizado de Java

## Sobre o Projeto

Este projeto explora o uso de Inteligência Artificial como ferramenta de apoio ao aprendizado, utilizando o NotebookLM para estruturar conhecimentos sobre a linguagem Java.

A proposta central foi desenvolver um “segundo cérebro” capaz de organizar informações, gerar insights e facilitar revisões de conteúdo.

---

##  Objetivos

- Consolidar conhecimentos iniciais em Java  
- Aplicar IA como ferramenta de aprendizado ativo  
- Desenvolver habilidades em engenharia de prompts  
- Criar um material reutilizável para estudos futuros  

---

##  Metodologia

O estudo foi conduzido com base em:

- Curadoria de conteúdos (artigos, PDF e vídeo)
- Interação com IA por meio de prompts estratégicos
- Análise das respostas geradas
- Refinamento contínuo das perguntas

---

##  Curadoria de Conteúdo

Foram utilizadas fontes abertas e acessíveis para construção do conhecimento:

- Artigos da DIO (fundamentos e carreira)
- Material acadêmico em PDF
- Conteúdo audiovisual para reforço prático

---

## Engenharia de Prompts

Durante o desenvolvimento, foram testadas diferentes abordagens de interação com a IA.

### Exemplos:
- Diferença entre JDK, JRE e JVM  
- Relevância do Java no mercado atual  
- Aplicações práticas da linguagem  

###  Aprendizados:
- A qualidade do prompt impacta diretamente na qualidade da resposta  
- Prompts específicos geram respostas mais técnicas  
- Iteração é essencial para aprofundamento  

---

##  Desafios Encontrados

- Limitação de conteúdos gratuitos  
- Respostas superficiais em alguns casos  
- Necessidade de refinamento constante dos prompts  

---
